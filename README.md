# Lista Dinâmica em C

Implementação de uma lista dinâmica onde se adiciona valores e se imprime a lista, além de um algoritmo de ordenação (bubble sort) para a mesma.

## Funções implementadas:
- put
- first
- list
- sort
- exit


## Uso

- Linux
    - Compilar com `make list`
    - Executar com `./list`
